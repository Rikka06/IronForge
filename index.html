1. 用户协议

**Summary:** 1.1 导言

淬火工坊（IronForge）是一款利用智能手表传感器检测臂部往复运动的辅助工具，帮助用户追踪频率、时长等数据，并提供评级与成就系统。安装或使用即视为同意本协议。

**Summary:** 1.2 服务说明

应用通过内置加速度计实时监测运动，提供SPM（每分钟冲程数）、总次数、时长统计，以及成就解锁功能。所有计算均在设备本地完成。

**Summary:** 1.3 用户行为规范
  • 仅在安全环境中使用，避免驾驶或操作重型机械时分心。
  • 禁止逆向工程、修改代码或用于非法目的。
  • 尊重知识产权，不得商业复制或分发。
**Summary:** 1.4 知识产权

应用代码、UI设计、图标、文案、评级算法及成就系统均归开发者所有，受著作权法保护。未经书面许可，不得改编或商用。

**Summary:** 1.5 协议修改

开发者保留修改权。修改后继续使用即视为接受新条款。我们建议定期查看更新。

2. 隐私政策

核心承诺：本应用无任何网络权限，所有数据100%本地存储，不上传、不分享、不用于AI训练或广告追踪。
**Summary:** 2.1 信息的收集
  • 传感器数据：实时读取三轴加速度计，用于计算SPM和冲程次数（不持久化保存）。
  • 本地数据：历史记录（时长、次数、评级）、设置（灵敏度、震动强度）、成就状态。
**Summary:** 2.2 信息的存储与安全

数据仅存于手表本地，无云端备份或传输。应用无联网代码，确保隐私零泄露。

**Summary:** 2.3 信息的使用

仅用于应用内显示，如报告、历史统计和成就解锁。不用于任何外部目的。

**Summary:** 2.4 信息的控制与删除

用户可随时删除单条或全部历史记录。卸载应用将永久清除所有数据。

3. 健康与免责声明

警告：本应用非医疗设备。过度运动可能导致伤害，请咨询医生并量力而行。
**Summary:** 3.1 运动健康风险

数据仅供娱乐参考，不能替代专业医疗建议。高频臂部运动可能引起肌肉疲劳、关节损伤或腱鞘炎。开发者不对因不当使用导致的健康问题负责。

**Summary:** 3.2 设备安全风险

确保手表固定，避免运动中脱落或碰撞。任何设备损坏或财产损失由用户自负。

**Summary:** 3.3 数据准确性

受传感器限制，可能有数据偏差。开发者不保证绝对准确。
