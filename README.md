## 🌏 English

### Overview
**IronForge** (淬火工坊) is a high-performance, professional digital forging simulator designed specifically for the **Xiaomi Vela OS** wearable ecosystem. By leveraging the advanced motion sensors of smartwatches, it provides an immersive feedback loop for physical activity, tracking every "strike" with precision.

### Key Features
- **🎯 Precision Tracking**: Real-time analysis of G-sensor and Gyroscope data to identify and count intense movements.
- **🏆 Achievement System**: Unlock various ranks and titles as you progress through your "forging" journey.
- **📊 Detailed Statistics**: Comprehensive history logs and statistical analysis of your daily performance.
- **⚡ Tactile Feedback**: Optimized vibration patterns for a realistic "impact" feel during use.
- **⚙️ Customization**: Adjustable sensitivity thresholds to match different user styles and environments.

### Technical Stack
- Built on **Xiaomi Vela Framework** (.ux components).
- Developed using **aiot-toolkit**.
- Optimized for low-power consumption on circular and square watches.

### Getting Started
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development mode:
   ```bash
   npm run start
   ```
4. Build for release:
   ```bash
   npm run build
   ```

## 🏮 简体中文

### 项目简介
**淬火工坊 (IronForge)** 是一款专为 **小米澎湃 OS (Xiaomi Vela)** 手表生态开发的专业数字锻造模拟器。通过调用智能手表的精密度传感器，它能够实时捕捉爆发性的肢体动作，并将其转化为虚拟世界中的每一次“击打”与“淬火”。

### 核心功能
- **🎯 精准捕捉**: 基于重力感应器 (G-Sensor) 与陀螺仪数据的实时算法分析，精准统计动作频次。
- **🏆 荣誉成就**: 完善的等级与成就系统，见证您的每一次自我突破。
- **📊 详尽统计**: 自动记录锻炼历史，提供多维度的统计报告与趋势分析。
- **⚡ 沉浸触感**: 深度优化的振动反馈算法，模拟真实的金属碰撞打击感。
- **⚙️ 灵敏调节**: 支持自定义触点灵敏度，适配多种运动强度与使用场景。

### 技术架构
- 基于 **Xiaomi Vela 框架** (.ux 规范) 开发。
- 使用 **aiot-toolkit** 进行构建与调试。
- 针对圆形及方形表盘进行了全面的 UI/UX 适配。

### 开发指南
1. 克隆本项目。
2. 安装依赖包：
   ```bash
   npm install
   ```
3. 开启实时开发预览：
   ```bash
   npm run start
   ```
4. 构建发布包：
   ```bash
   npm run build
   ```
